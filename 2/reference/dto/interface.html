classmethod on_registration(parsed_type: ParsedType, route_handler: BaseRouteHandler) → None#

Receive the parsed_type and route_handler that this DTO is configured to represent.

At this point, if the DTO type does not support the annotated type of parsed_type, it should raise an UnsupportedType exception.

Parameters:

  • parsed_type – ParsedType instance, will be either the parsed annotation of a "data" kwarg, or the parsed return type annotation of a route handler.

  • route_handler – HTTPRouteHandler DTO type is declared upon.

Raises:

UnsupportedType – If the DTO type does not support the annotated type of parsed_type.
